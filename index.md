# Support — Mandaring Flashcards

If you need help, found a bug, or want to request a feature, contact us:

- **Email:** thiessen[dot]james[at]gmail[d0t]com

## Helpful info to include
When you email, please include:
- Device model (e.g., iPhone 15)
- iOS version
- App version
- What you expected vs what happened

## FAQ
### Restore purchases
Open the app → Settings → Restore Purchases (if available).

### Report a bug
Send a short description + steps to reproduce, and screenshots if possible.
